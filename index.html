Online Presence

Indeed Web Crawler

This is a simple CRUD application that crawls indeed.com for front-end engineer job postings using Node.js and Angular.js. Each job title you see in the screen shot is the "title" and "URL" pulled from indeed.com which the user can click through to and apply. The user can also add notes to each job entry once added to the job list and update those notes as they go. Here's the repository for your reference.

I used Heroku to host and deploy this application here.
